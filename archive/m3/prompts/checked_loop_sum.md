# Task: Checked Loop Sum

Write a complete program in the target language.

Requirements:

- Initialize `i = 0` and `sum = 0`.
- Loop while `i < 5`.
- Add `i` to `sum` using the language's checked or trapping integer operation.
- Increment `i` using the language's checked or trapping integer operation.
- After the loop, verify `sum == 10`.
- Compute `doubled = sum * 2` using checked or trapping integer arithmetic.
- Verify `doubled == 20`.
- Exit successfully only if all checks pass.

Do not use unchecked overflow, undefined behavior, or an unsafe escape hatch.
