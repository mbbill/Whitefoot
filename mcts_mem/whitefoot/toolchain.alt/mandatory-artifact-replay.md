- One trusted semantic kernel owns source acceptance and artifact replay judgments.
- The originating invocation projects one canonical artifact and completely replays only those bytes through the same kernel before lowering authority exists.
- Lowering-visible state is reconstructed from replayed bytes; producer drafts are discarded.
- Later, cached, third-party, and hand-authored artifact bytes are audit-only and cannot construct lowering authority.
- Same-kernel replay is not independent semantic evidence; independent grammar, source/tree, conformance, model, target, backend, guard, publication, and optional-fact lanes remain required.
